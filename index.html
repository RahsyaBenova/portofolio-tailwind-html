Smart Water Meter

This project is an AIoT-based water control system that uses the ESP32 module. The system is designed to accurately measure water flow and automatically control the solenoid valve. By using a water flow sensor, the system can monitor water usage in real-time and transmit the data to an IoT platform for further analysis. In addition, the system can be integrated with solar panels to support operations without using conventional power sources.

Digital Clock

This project is a digital clock that uses an ESP32 module to retrieve time data from the internet. It displays the time and date in real-time with high precision, as the time is synchronized with a global time server via a Wi-Fi connection. The large and bright time display makes it ideal for use in public spaces or rooms that require clear visibility of the time.

Close

Smart Trash Bin

This project is a smart trash bin that uses Arduino Uno to control servo motors. The system is designed so that the bin lid can open and close automatically when it detects an object nearby. This technology makes it easier for users to dispose of garbage without having to touch the bin lid, thus improving hygiene and convenience.
